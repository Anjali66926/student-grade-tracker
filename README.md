# 🎓 Student Grade Tracker

This is a Java-based console application developed as part of my CodeAlpha Internship.

## 🚀 Features
- Add student details (Name & Marks)
- Calculate:
  - Average marks
  - Highest marks
  - Lowest marks
- Automatic Grade Calculation (A/B/C/F)
- Exception Handling for invalid inputs
- Menu-driven user interface

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Exception Handling

## ▶️ How to Run

1. Compile the program:
   javac *.java

2. Run the program:
   java Main
